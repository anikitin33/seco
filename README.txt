Willkommen zu Ihrem Repository Programmieren in Secure Coding

Das Repository ist initialisiert mit einem Ordner für

    ueb00

so wie Sie auch die anderen Ordner je Übungsblatt benennen sollten.

Auf der Konsole (oder halt mit klicken) checken Sie das Repository aus mit
    git clone https://scm.inftech.hs-mannheim.de/scm/git/20secoxx
wobei xx Ihre persönliche Nummer ist.

Sie werden in Ordnern oft ein Makefile haben und darin C oder C++ Code
erstellen, selten mal auch ein Python-File oder später Rust, wenn es gut läuft.
